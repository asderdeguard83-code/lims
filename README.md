# Управление техническими требованиями

Веб-приложение для управления техническими требованиями к материалам на основе нормативных документов (ГОСТ).

## 🚀 Функциональность

### Основные возможности
- **Управление продуктами** - создание, редактирование, удаление продуктов
- **Наборы требований** - работа с коллекциями продуктов и их характеристик
- **Технические показатели** - детальное управление параметрами материалов
- **Нормативные документы** - привязка к ГОСТам и другим стандартам

### Интерфейс
- **Левая панель** - управление продуктами (поиск, фильтрация, добавление/удаление)
- **Центральная область** - отображение характеристик и показателей выбранного продукта
- **Верхняя панель** - управление наборами требований

## 🛠 Технологический стек

### Frontend
- **Vue 3** - основной фреймворк
- **TypeScript** - типизация
- **Pinia** - управление состоянием
- **Vue Router** - маршрутизация
- **Vite** - сборка и разработка
- **SCSS** - стилизация

## 📦 Установка и запуск

### Предварительные требования
- Node.js 16+ 
- npm или yarn

### Установка
```bash
# Клонирование репозитория
git clone <repository-url>
cd technical-requirements-app

# Установка зависимостей
npm install
Запуск в режиме разработки
bash
npm run dev
Приложение будет доступно по адресу: http://localhost:3000

Сборка для production
bash
npm run build
Просмотр собранной версии
bash
npm run preview
